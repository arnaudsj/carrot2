Welcome to Carrot2 Document Clustering Workbench! You can use it to test Carrot2 clustering on web search results or your own data.

To run Carrot2 clustering on web search results:

  1. Type your search query in the Query field on the left
  2. Press the Process button

For further instructions, please see Carrot2 Manual:

  • Clustering plain text, HTML and MS Word documents
  • Clustering documents from XML files
  • Clustering documents from XML feeds
  • Clustering documents from a Lucene index
  • Clustering documents from a Solr index
  • Tuning clustering in Carrot2 Document Clustering Workbench
